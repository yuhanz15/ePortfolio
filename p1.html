Vaccine Reservation System

As part of my work in CSE 414 (Database Systems) at the University of Washington, I built a small vaccine reservation system using Java and SQL. The project focused on designing a working backend where patients could create accounts, book vaccine appointments, and even select specific caregivers if needed.

Caregivers, on the other hand, could manage their available schedules and track vaccine inventories through the system. To make the user experience smoother, I added basic error handling to guide users through common issues they might encounter during reservations.
